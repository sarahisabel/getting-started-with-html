The Basics of the Web

The World Wide Web
- collection of HTML documents
- invented in 1990s
- you can find all types of files on the web (plain text, images videos, etc.)

The basics: HTML, URLs, HTTP, Web Applications

Major pieces
YOU (your computer & Browser)
The Internet
Servers

HTML - HyperText Markup Language
HTML documents are the heart of the web.

Made up of:
- text content (what you see)
- text markup (what it looks like)
- references to other documents (images, videos, etc)
- links to other pages

HTML Markup - Tags
- always have an opening and closing tag (the name is always the same)
- Attributes like 'a href' (a is tag, href is link)
- whitespace can be affected by inline or block tags

Inline tags:
b tag - bold
em tag - emphasis (italics)
br tag - break
a tag - link
img tag - image
span

Block tags:
p tag - paragraph
divs
